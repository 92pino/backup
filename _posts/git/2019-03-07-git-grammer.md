## Terminal 단축키


> 주로 사용할것이라고 생각하는 terminal 명령어를 정리


- ls : 해당 디렉토리의 폴더 목록을 조회<br>

```
ls 현재 디렉토리명
ls desktop
```

- cd : 이동하고 싶은 디렉토리로 경로 이동<br>

```
cd 이동하고 싶은 디렉토리 경로
cd desktop
```

- pwd : 현재 디렉토리의 경로 확인<br>

```
pwd 현재 디렉토리 경로
pwd desktop
```

- mkdir : 디렉토리 생성<br>

```
mkdir 디렉토리명
mkdir testFolder
```

- rmdir : 디렉토리 삭제<br>

```
mkdir 디렉토리명
mkdir testFolder
```

- touch : 파일 생성<br>

```
touch 파일명
touch testFile.txt
```

- cp : 파일 복사 (폴더 복사는 안됨)<br>

```
cp 파일명
cp testFile.txt
```

- mv : 파일 이동<br>

```
mv 파일명 파일 이동 경로
mv testFile.txt ../desktop
```

- rm : 파일 삭제<br>

```
rm 파일명
rm testFile.txt
```

- clear : 터미널의 셀 화면을 클리어<br>

```
clear
```

- exit : 화면 종료<br>

```
exit
```
