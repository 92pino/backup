✔︎ Xcode 단축키
  - 주로 사용하는 단축키들
    - Cmd + option + enter    :  화면 분할
    - Cmd + enter             :  화면 합치기
    - Cmd + R                 :  시뮬레이터 실행
    - Cmd + Shift + H         :  시뮬레이터에서 홈화면으로 가는 단축키
    - Cmd + K                 :  로그창 clear
    - 단축키는 더 많지만 나중에 추가적으로 정리 필요할듯..
